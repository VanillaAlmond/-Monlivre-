# -Monlivre-
Site web public statique présentant la solution digitale de simplification du paiement de l'impôt synthétique à Madagascar, basée sur l'amélioration du service Hetraphone. Le site inclut une page d'accueil, une présentation des fonctionnalités clés, un accès au code source pour vérification, et une visibilité publique sécurisée via GitHub Pages.
